# ☁️ 30 Serviços AWS Mais Cobrados nas Certificações

Vou listar os **30 serviços da AWS mais cobrados nas certificações**, especialmente nas provas:

* AWS Certified Cloud Practitioner
* AWS Certified Solutions Architect – Associate
* AWS Certified Developer – Associate

Eles aparecem constantemente nas questões de arquitetura, cenários de aplicação e perguntas conceituais.

---

## 🖥️ Compute (Execução de aplicações)

### 1. Amazon EC2

Máquinas virtuais na AWS.

Muito cobrado em:

* escalabilidade
* auto scaling
* tipos de instância
* pricing (on-demand, reserved, spot)

---

### 2. AWS Lambda

Execução de código **serverless**.

Cai muito em perguntas sobre:

* arquitetura sem servidor
* processamento de eventos
* integração com outros serviços.

---

### 3. Amazon ECS

Orquestração de containers Docker.

---

### 4. Amazon EKS

Kubernetes gerenciado pela AWS.

---

### 5. AWS Fargate

Executa containers sem gerenciar servidores.

---

# 🗄️ Storage (Armazenamento)

### 6. Amazon S3

Um dos serviços **mais cobrados**.

Cai em temas como:

* storage de arquivos
* static website
* backup
* versionamento
* lifecycle

---

### 7. Amazon EBS

Discos anexados ao EC2.

---

### 8. Amazon EFS

File system compartilhado entre servidores.

---

### 9. Amazon S3 Glacier

Armazenamento de **backup de longo prazo**.

---

# 🗃️ Databases

### 10. Amazon RDS

Banco relacional gerenciado.

Suporta:

* MySQL
* PostgreSQL
* MariaDB
* SQL Server
* Oracle

---

### 11. Amazon Aurora

Versão otimizada do RDS.

Alta performance.

---

### 12. Amazon DynamoDB

Banco **NoSQL serverless**.

Muito cobrado nas provas.

---

### 13. Amazon ElastiCache

Cache em memória.

Suporta:

* Redis
* Memcached

---

# 🌐 Networking

### 14. Amazon VPC

Rede virtual da AWS.

Muito cobrado em:

* subnets
* route tables
* internet gateway
* NAT gateway

---

### 15. Elastic Load Balancing

Distribui tráfego entre servidores.

Tipos:

* ALB
* NLB
* CLB

---

### 16. Amazon Route 53

DNS da AWS.

Cai muito em:

* failover
* latency routing
* health checks

---

### 17. Amazon CloudFront

CDN da AWS.

Distribuição global de conteúdo.

---

### 18. AWS Direct Connect

Conexão dedicada entre empresa e AWS.

---

# 🔐 Segurança

### 19. AWS IAM

Gerenciamento de identidade e acesso.

Muito cobrado.

Principais conceitos:

* users
* roles
* policies
* least privilege

---

### 20. AWS KMS

Gerenciamento de chaves criptográficas.

---

### 21. AWS WAF

Firewall para aplicações web.

---

### 22. AWS Shield

Proteção contra ataques DDoS.

---

# 📊 Monitoramento

### 23. Amazon CloudWatch

Monitoramento de recursos.

Inclui:

* métricas
* logs
* alarmes

---

### 24. AWS CloudTrail

Auditoria de atividades na AWS.

Registra chamadas de API.

---

# 🔄 Integração e Mensageria

### 25. Amazon SQS

Fila de mensagens.

Arquiteturas desacopladas.

---

### 26. Amazon SNS

Sistema de notificações.

Modelo **pub/sub**.

---

### 27. Amazon EventBridge

Event bus para integração de eventos.

---

# 🧩 Serverless

### 28. Amazon API Gateway

Criação de APIs REST ou HTTP.

Muito usado com Lambda.

---

### 29. AWS Step Functions

Orquestração de workflows serverless.

---

# 🛠 Infraestrutura como código

### 30. AWS CloudFormation

Permite criar infraestrutura usando **templates**.

Infraestrutura como código.
